#### 🛡️ ARES — Semi-automated pentesting platform *(Final Degree Project)*
Full stack web app with a 4-layer architecture: React frontend → Node.js/Express backend → custom C binaries → system tools (Nmap, Metasploit, SSH).
- Custom C binaries for network scanning (CIDR) and connection handling
